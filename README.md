## 🏛️ 第一章：核心哲學與定位 (Philosophy)

本系統基於 **「天使號令·光之聖典」** 架構，並深度整合 **RAGFlow** 開源技術，實現企業級 ESG 知識的精準召回與自動化治理。

- **DeepDoc 解析：** 採用 RAGFlow 的深度佈局分析技術（MinerU / Docling），確保複雜 ESG 報告（如 IFRS S1/S2）中的表格、圖表與文本能被 100% 精準識別。
- **Agentic RAG：** 超越傳統單向 RAG，建立具備「記憶」與「規劃」能力的 Agent 鏈路，將數據處理轉化為可觀察、可調整的 Agentic Workflow。
- **王道治理 (Wangdao)：** 以利他精神為演算法核心，將 RAG 產出的洞察對標全球 500 強最佳實踐。

## 📊 第三章：領袖指標 (KPI Widgets)
